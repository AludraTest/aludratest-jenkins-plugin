To generate HTML log files, you must configure your AludraTest project to use the Log4Testing org.aludratest.log4testing.html.HtmlTestLogWriter writer. Note that you should configure Jenkins to archive the HTML logs (e.g. generated to target/result/html). Here, specify where the HTML log files are archived to, relative to the build URL, e.g. artifact/target/result/html. This will enable the AludraTest Jenkins Plugin to link to the HTML log of each test case.
